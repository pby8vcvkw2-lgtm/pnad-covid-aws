# PNAD COVID - Pipeline de Dados na AWS

Projeto desenvolvido para o Tech Challenge da fase 3 na FIAP.

Objetivo do projeto:

Construir um pipeline de dados utilizando dados da PNAD COVID, aplicando etapas de:

- Ingestão de dados
- Tratamento
- Armazenamento em nuvem
- Análise exploratória

Tecnologias utilizadas:

- Python
- Pandas
- AWS
- Git
- GitHub
