# PNAD COVID - Pipeline de Dados na AWS

Projeto desenvolvido para o Tech Challenge da fase 3 na FIAP.

Objetivo do projeto:

Construir um pipeline de dados utilizando dados da PNAD COVID, aplicando etapas de:

- Ingestão de dados
- Tratamento
- Armazenamento em nuvem
- Análise exploratória

Tecnologias utilizadas:

- Python
- Pandas
- AWS
- Git
- GitHub

## O que contém em cada pasta

- data/raw: arquivos com os dados brutos.
- data/processed: arquivos com os dados limpos e tratados para a análise.
- notebooks: EDA (Análise exploratória de dados).
- src: Scripts em python.
- sql: Scripts SQL.
- docs: Documentação adicional.
